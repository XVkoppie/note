# CDN内容分发

CDN，内容分发网络，它的核心思想非常直观：把我网站的资源（比如图片、视频、CSS/JS文件）提前拷贝到全国各地、甚至全球各地的缓存服务器上。当用户想访问我的网站时，就不需要千里迢迢都跑到我唯一的主服务器来取数据了，而是直接由离他最近的那台缓存服务器把数据给他。
1. 加速用户访问：这是最直接的，数据离用户更近，延迟更低，网页打开更快。

2. 减轻源站压力：绝大多数请求都被边缘节点消化了，源服务器带宽和计算压力骤减，不容易被高并发流量打垮。

3. 提升可用性：即使源服务器暂时出现故障，或者网络出现波动，由于内容分布在大量节点上，用户仍然可以从其他健康的节点获取到缓存的内容，网站的容灾能力更强。”
